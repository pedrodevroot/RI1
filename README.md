# RI1 - Técnicas de Programação

Projeto desenvolvido para a disciplina de Técnicas de Programação – FATEC.

## 📌 Descrição

Este projeto foi desenvolvido utilizando JavaScript puro, com foco em lógica de programação e manipulação de estruturas de dados.

## 🚀 Como executar o projeto

### 1️⃣ Pré-requisitos

Ter o Node.js instalado na máquina.

No terminal, dentro da pasta do projeto, execute:

node nome-do-arquivo.js
